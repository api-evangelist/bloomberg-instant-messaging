# Bloomberg Instant Messaging

Bloomberg Instant Messaging (IB) is a secure, compliant messaging platform embedded in the Bloomberg Terminal and accessible via Bloomberg Anywhere. It enables real-time communication between financial professionals globally with archiving and compliance monitoring.

## APIs

- **Bloomberg IB Messaging API** - Programmatic access to Bloomberg's IB messaging network
- **Bloomberg B-Chat** - Enterprise communication platform for broader connectivity
